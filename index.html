Devito

The easiest way to deploy your own url-shortener using Elixir

What?

Devito is open-source software to make your own custom domain short links.

Instead of passing around long, hard to remember links like www.youtube.com/watch?v=dQw4w9WgXcQ,
you can make your own short links like cool.dev/abc.

How?

Devito is written in Elixir. As such, one of the best and easiest deployment options is to use Gigalixir. You'll need to setup an account on gigalixir.com, set one environment variable, then deploy. It's that easy.

Step-by-step instructions are available on Github.
